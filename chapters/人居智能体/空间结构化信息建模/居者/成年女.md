# 人口学 · 成年女

> [总览 · Occupant](../居者（Occupant）.md) | 与[成年男](./成年男.md)同模式；[站](../posture/站.md) / [坐](../posture/坐.md) 为姿态层约定。图中体块色仅作区分，**元数据** 以 `sex=女` 等字段为准。

- 语义：`OCCUPANT` + `age_group=adult` + `sex=女` + `posture=stand|sit`。

| 与 posture 搭配 | 肩宽（m，例） | 身厚（m，例） | 高（m，z=0 起） | 说明 |
|------|------------|------------|----------------|------|
| 站 | 0.44 | 0.24 | 1.62 | 可改标准分位 |
| 坐 | 0.47 | 0.60 | 1.12 | 同成年男之坐姿说明 |

**创建步骤** 同[成年男](./成年男.md)，仅替换 (s, t, h) 与标注。

```python
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d.art3d import Poly3DCollection
import numpy as np

L, W, H_room = 4.2, 3.6, 3.0
t = 0.2
F_STAND = (0.44, 0.24, 1.62)
F_SIT = (0.47, 0.60, 1.12)
stand_cx, stand_cy = L * 0.35, W * 0.50
sit_cx, sit_cy = L * 0.58, W * 0.42


def cuboid(x, y, z, dx, dy, dz):
    x1, y1, z1 = x + dx, y + dy, z + dz
    return [
        [(x, y, z), (x1, y, z), (x1, y1, z), (x, y1, z)],
        [(x, y, z1), (x1, y, z1), (x1, y1, z1), (x, y1, z1)],
        [(x, y, z), (x1, y, z), (x1, y, z1), (x, y, z1)],
        [(x1, y, z), (x1, y1, z), (x1, y1, z1), (x1, y, z1)],
        [(x1, y1, z), (x, y1, z), (x, y1, z1), (x1, y1, z1)],
        [(x, y1, z), (x, y, z), (x, y, z1), (x, y1, z1)],
    ]


def flatten(faces):
    return [np.array(f) for f in faces if np.array(f).ndim == 2]


def add(ax, faces, color, alpha=0.45):
    ax.add_collection3d(
        Poly3DCollection(
            faces,
            facecolors=color,
            edgecolors="black",
            linewidths=0.3,
            alpha=alpha,
        )
    )


def person_box(s, t, h, cx, cy):
    x0, y0 = cx - s / 2, cy - t / 2
    return cuboid(x0, y0, 0, s, t, h), (x0, y0)


wall_faces = []
wall_faces.extend(cuboid(L - t / 2, 0, 0, t, W, H_room))
wall_faces.extend(cuboid(-t / 2, 0, 0, t, W, H_room))
wall_faces.extend(cuboid(0, W - t / 2, 0, L, t, H_room))
wall_faces.extend(cuboid(0, -t / 2, 0, L, t, H_room))
floor_faces = cuboid(0, 0, -t, L, W, t)
stand_faces, (sx0, sy0) = person_box(*F_STAND, stand_cx, stand_cy)
sit_faces, (jx0, jy0) = person_box(*F_SIT, sit_cx, sit_cy)

fig = plt.figure(figsize=(8, 6))
ax = fig.add_subplot(111, projection="3d")
add(ax, flatten(wall_faces), (0.2, 0.6, 1, 0.2), alpha=0.2)
add(ax, flatten(floor_faces), (0.7, 0.7, 0.7, 1.0), alpha=1.0)
add(ax, flatten(stand_faces), (0.75, 0.45, 0.85, 0.5), alpha=0.45)
add(ax, flatten(sit_faces), (0.55, 0.35, 0.7, 0.5), alpha=0.45)
ax.text(sx0 + F_STAND[0] / 2, sy0 + F_STAND[1] / 2, F_STAND[2] * 0.9, "adult 女·站", color="k", fontsize=9)
ax.text(jx0 + F_SIT[0] / 2, jy0 + F_SIT[1] / 2, F_SIT[2] * 0.9, "adult 女·坐", color="k", fontsize=9)
ax.view_init(elev=22, azim=120)
ax.set_xlim(0, L)
ax.set_ylim(0, W)
ax.set_zlim(0, H_room)
ax.set_box_aspect([L, W, H_room])
ax.set_xlabel("x")
ax.set_ylabel("y")
ax.set_zlabel("z")
plt.tight_layout()
plt.show()
```
