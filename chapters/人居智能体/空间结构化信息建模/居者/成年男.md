# 人口学 · 成年男

> 本页在 **demographic** 下给出**默认体尺**与示例代码；**立姿 / 坐姿** 的几何与语义见 [../posture/站.md](../posture/站.md)、[../posture/坐.md](../posture/坐.md)。总览 [../居者（Occupant）.md](../居者（Occupant）.md)。

- **立姿**与**坐姿**在几何上均用**轴对齐长方体**（`cuboid`）表达，差别在于**默认人体测量学尺寸**与 `posture` 字段。坐姿体块可视为**「座椅上人体」的外包**（自楼板 $z=0$ 起，进深含膝前，高为**地面到头顶**）；不单独建椅子时也可用于粗算。与椅分体时见 [../../功能构件/椅子.md](../../功能构件/椅子.md)。
- 语义建议：`OCCUPANT` + `demographic.age_group=adult` + `sex=男` + `posture=stand|sit`；`activity` 在行为层再绑。

| 与 posture 搭配 | 肩宽-左右向（m，例） | 身厚-前后向（m，例） | 包络高（m，自 z=0） | 说明 |
|------|-------------------|-------------------|------------------------|------|
| 站 | 0.46 | 0.25 | 1.75 | 可改国标分位 |
| 坐 | 0.50 | 0.64 | 1.22 | 臀后—膝前、地面—头顶外包 |

**立姿 / 坐姿** 落点：以平面参考 (cx, cy) 用 `cuboid(x0,y0,0,s,t,h)`，其中 x0=cx-s/2, y0=cy-t/2。

```python
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d.art3d import Poly3DCollection
import numpy as np

L, W, H_room = 4.2, 3.6, 3.0
t = 0.2

# 成年男：立 / 坐 单箱 (s, t, h)，m
M_STAND = (0.46, 0.25, 1.75)
M_SIT = (0.50, 0.64, 1.22)
stand_cx, stand_cy = L * 0.38, W * 0.40
sit_cx, sit_cy = L * 0.60, W * 0.48


def cuboid(x, y, z, dx, dy, dz):
    x1, y1, z1 = x + dx, y + dy, z + dz
    return [
        [(x, y, z), (x1, y, z), (x1, y1, z), (x, y1, z)],
        [(x, y, z1), (x1, y, z1), (x1, y1, z1), (x, y1, z1)],
        [(x, y, z), (x1, y, z), (x1, y, z1), (x, y, z1)],
        [(x1, y, z), (x1, y1, z), (x1, y1, z1), (x1, y, z1)],
        [(x1, y1, z), (x, y1, z), (x, y1, z1), (x1, y1, z1)],
        [(x, y1, z), (x, y, z), (x, y, z1), (x, y1, z1)],
    ]


def flatten(faces):
    return [np.array(f) for f in faces if np.array(f).ndim == 2]


def add(ax, faces, color, alpha=0.45):
    ax.add_collection3d(
        Poly3DCollection(
            faces,
            facecolors=color,
            edgecolors="black",
            linewidths=0.3,
            alpha=alpha,
        )
    )


def person_box(s, t, h, cx, cy):
    x0, y0 = cx - s / 2, cy - t / 2
    return cuboid(x0, y0, 0, s, t, h), (x0, y0)


wall_faces = []
wall_faces.extend(cuboid(L - t / 2, 0, 0, t, W, H_room))
wall_faces.extend(cuboid(-t / 2, 0, 0, t, W, H_room))
wall_faces.extend(cuboid(0, W - t / 2, 0, L, t, H_room))
wall_faces.extend(cuboid(0, -t / 2, 0, L, t, H_room))
floor_faces = cuboid(0, 0, -t, L, W, t)

stand_faces, (sx0, sy0) = person_box(*M_STAND, stand_cx, stand_cy)
sit_faces, (jx0, jy0) = person_box(*M_SIT, sit_cx, sit_cy)

fig = plt.figure(figsize=(8, 6))
ax = fig.add_subplot(111, projection="3d")
add(ax, flatten(wall_faces), (0.2, 0.6, 1, 0.2), alpha=0.2)
add(ax, flatten(floor_faces), (0.7, 0.7, 0.7, 1.0), alpha=1.0)
add(ax, flatten(stand_faces), (0.95, 0.75, 0.2, 0.55), alpha=0.45)
add(ax, flatten(sit_faces), (0.85, 0.55, 0.1, 0.55), alpha=0.45)
ax.text(sx0 + M_STAND[0] / 2, sy0 + M_STAND[1] / 2, M_STAND[2] * 0.9, "adult 男·站", color="k", fontsize=9)
ax.text(jx0 + M_SIT[0] / 2, jy0 + M_SIT[1] / 2, M_SIT[2] * 0.9, "adult 男·坐", color="k", fontsize=9)
ax.view_init(elev=22, azim=120)
ax.set_xlim(0, L)
ax.set_ylim(0, W)
ax.set_zlim(0, H_room)
ax.set_box_aspect([L, W, H_room])
ax.set_xlabel("x")
ax.set_ylabel("y")
ax.set_zlabel("z")
plt.tight_layout()
plt.show()
```
