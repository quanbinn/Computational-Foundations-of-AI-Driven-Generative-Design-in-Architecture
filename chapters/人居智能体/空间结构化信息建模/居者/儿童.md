# 人口学 · 儿童（3 / 6 / 10 岁）

> [总览 · Occupant](../居者（Occupant）.md) | `demographic: child` + `age_years`；[站](../posture/站.md) / [坐](../posture/坐.md) 定义 `posture`。

- 儿童按**年龄组**与**立/坐**区分；几何为 `cuboid`，例：`"age_years": 3` 且 `posture: stand|sit`。
- 下表为**教学用**近似 (s, t, h)，可换《7 岁以下儿童生长发育参照标准》等 P 分位。坐姿单箱为地面—头顶、含膝前进深之**外包**；幼儿椅高方差大，可拆体块（进 [`posture/坐`](../posture/坐.md)）。

| 年龄 | 立（s, t, h, m 例） | 坐（s, t, h, m 例） | 注 |
|------|-------------------|-------------------|-----|
| 3 岁 | 0.32, 0.24, 0.98 | 0.32, 0.50, 0.78 | 项目以标准表为准 |
| 6 岁 | 0.36, 0.26, 1.18 | 0.36, 0.55, 0.94 | 同上 |
| 10 岁 | 0.40, 0.28, 1.40 | 0.40, 0.58, 1.08 | 方差大 |

**创建步骤：** 选年龄行与姿态，取 (s, t, h)，(cx, cy) 上 `person_box`/`cuboid`；绑定 `age_years` 与 `posture`。

```python
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d.art3d import Poly3DCollection
import numpy as np

L, W, H_room = 5.0, 4.0, 3.0
t = 0.2

KID = {
    3: {"stand": (0.32, 0.24, 0.98), "sit": (0.32, 0.50, 0.78)},
    6: {"stand": (0.36, 0.26, 1.18), "sit": (0.36, 0.55, 0.94)},
    10: {"stand": (0.40, 0.28, 1.40), "sit": (0.40, 0.58, 1.08)},
}


def cuboid(x, y, z, dx, dy, dz):
    x1, y1, z1 = x + dx, y + dy, z + dz
    return [
        [(x, y, z), (x1, y, z), (x1, y1, z), (x, y1, z)],
        [(x, y, z1), (x1, y, z1), (x1, y1, z1), (x, y1, z1)],
        [(x, y, z), (x1, y, z), (x1, y, z1), (x, y, z1)],
        [(x1, y, z), (x1, y1, z), (x1, y1, z1), (x1, y, z1)],
        [(x1, y1, z), (x, y1, z), (x, y1, z1), (x1, y1, z1)],
        [(x, y1, z), (x, y, z), (x, y, z1), (x, y1, z1)],
    ]


def flatten(faces):
    return [np.array(f) for f in faces if np.array(f).ndim == 2]


def add(ax, faces, color, alpha=0.45):
    ax.add_collection3d(
        Poly3DCollection(
            faces,
            facecolors=color,
            edgecolors="black",
            linewidths=0.3,
            alpha=alpha,
        )
    )


def person_box(s, t, h, cx, cy):
    x0, y0 = cx - s / 2, cy - t / 2
    return cuboid(x0, y0, 0, s, t, h), (x0, y0)


wall_faces = []
wall_faces.extend(cuboid(L - t / 2, 0, 0, t, W, H_room))
wall_faces.extend(cuboid(-t / 2, 0, 0, t, W, H_room))
wall_faces.extend(cuboid(0, W - t / 2, 0, L, t, H_room))
wall_faces.extend(cuboid(0, -t / 2, 0, L, t, H_room))
floor_faces = cuboid(0, 0, -t, L, W, t)

centers = {
    3: ((1.0, 0.7), (1.0, 2.1)),
    6: ((2.2, 0.7), (2.2, 2.1)),
    10: ((3.4, 0.7), (3.4, 2.1)),
}
colors_stand = {3: (0.2, 0.75, 0.85, 0.55), 6: (0.2, 0.55, 0.9, 0.55), 10: (0.3, 0.35, 0.85, 0.55)}
colors_sit = {3: (0.85, 0.5, 0.2, 0.5), 6: (0.9, 0.4, 0.35, 0.5), 10: (0.75, 0.35, 0.65, 0.5)}

fig = plt.figure(figsize=(9, 6.5))
ax = fig.add_subplot(111, projection="3d")
add(ax, flatten(wall_faces), (0.2, 0.6, 1, 0.2), alpha=0.18)
add(ax, flatten(floor_faces), (0.7, 0.7, 0.7, 1.0), alpha=1.0)

for age, (c_st, c_si) in centers.items():
    st_f, p0 = person_box(*KID[age]["stand"], *c_st)
    si_f, p1 = person_box(*KID[age]["sit"], *c_si)
    add(ax, flatten(st_f), colors_stand[age], alpha=0.42)
    add(ax, flatten(si_f), colors_sit[age], alpha=0.42)
    s, ti, h = KID[age]["stand"]
    ax.text(p0[0] + s / 2, p0[1] + ti / 2, h * 0.92, f"child {age}y·站", color="k", fontsize=8)
    s, ti, h = KID[age]["sit"]
    ax.text(p1[0] + s / 2, p1[1] + ti / 2, h * 0.92, f"child {age}y·坐", color="k", fontsize=8)

ax.view_init(elev=24, azim=115)
ax.set_xlim(0, L)
ax.set_ylim(0, W)
ax.set_zlim(0, H_room)
ax.set_box_aspect([L, W, H_room])
ax.set_xlabel("x")
ax.set_ylabel("y")
ax.set_zlabel("z")
plt.tight_layout()
plt.show()
```
