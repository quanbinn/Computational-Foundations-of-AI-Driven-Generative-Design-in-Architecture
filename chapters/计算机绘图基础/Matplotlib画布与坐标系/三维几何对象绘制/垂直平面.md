# 垂直平面

### x常值平面

```python
import numpy as np
from matplotlib import pyplot as plt

# ---------------- 3D坐标系初始化 ----------------
ax = plt.figure().add_subplot(111, projection='3d')

# ---------------- 生成一维采样点 ----------------
x = np.linspace(0, 9, 2)
y = np.linspace(0, 9, 2)

# ---------------- 构造二维网格 ----------------
X, Y = np.meshgrid(x, y)

print(X)
print(Y)

# ---------------- 绘制方法 ----------------
#  渲染出 x=1.5 的部分平面
ax.plot_surface(
    X = X * 0 + 1.5,   # x方向固定为常数1.5
    Y = Y,             # y方向展开
    Z = X              # z方向使用原X形成高度变化
)

# ---------------- 显示图形 ----------------
plt.show()
```
